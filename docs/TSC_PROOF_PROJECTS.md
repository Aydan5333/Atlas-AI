# TSC Proof Projects

## Purpose

TSC needs proof before chasing bigger rooms.

The rule:

> Your in is not clout. Your in is a prototype.

This document defines the first proof projects that should make TSC look organized, useful, and real before contacting bigger circles.

## Proof Project 001: TSC Culture Garage 001

### Core idea

A small private proof event showing the TSC formula:

- cars
- audio
- fashion
- live art
- music
- sim racing
- Atlas event operations
- education/storytelling

Suggested size: 20 to 40 people.

### Goal

Create a clean proof package that can be sent to local sponsors, CNM/UNM circles, production contacts, education contacts, and future event collaborators.

### Minimum viable setup

| Element | Minimum version |
| --- | --- |
| Anchor car | One trusted local build with clear permission |
| Audio feature | Explain the system, demo reactions, document the build |
| Live art | Paint/art on a panel, hood, helmet, glass, metal, or speaker box |
| Fashion | TSC interlocked logo cap, Pro Club tee, sticker pack, Culture Garage 001 sample |
| Sim station | Assetto Corsa, Forza, F1, or basic racing rig with leaderboard |
| Atlas station | QR signup, event checklist, build notes, schedule, recap prompts |
| Content | 60-second reel, 10 strong photos, founder video, sponsor recap sheet |

### What makes it different

This is not a car meet.
This is not a clothing pop-up.
This is not a party.

It is a small culture-to-skill proof event.

People should leave thinking:

> I came for cars, music, fashion, and art, but I saw a path into design, engineering, fabrication, business, and community.

## Proof Project 002: Auto / Audio / Fabrication Proof Lane

### Why this matters

The auto pillar should prove that TSC understands builds as culture, engineering, craft, documentation, and education.

This lane should not depend on one named person, one vehicle, or one sponsor connection. It should be flexible enough to work with any trusted local build that gives permission and fits the mission.

### Deliverables

- build story interview
- clean photos of the vehicle or system
- audio or fabrication overview
- tool/process documentation
- problem-solution breakdown
- 60-second feature reel
- sponsor-safe recap one-pager
- future TSC Audio / Fabrication concept

### Tone

Do not step on someone else's lane.
Do not imply a sponsorship or partnership that has not been approved.

Position TSC as:

> We help document, organize, and amplify real builders, real process, and real skill.

## Proof Project 003: TSC Academy Micro-Workshop

### Goal

Test the education side before calling it a school.

### Format

One 60 to 90 minute workshop connected to Culture Garage 001.

Possible first topics:

- how sim racing connects to vehicle dynamics
- how car audio enclosures connect to fabrication and math
- how a hat/shirt design becomes a product
- how live art becomes brand identity
- how Atlas organizes a project from idea to recap

### Output

- workshop outline
- short handout
- photos/video
- student/member feedback
- next workshop topic list

## Proof Project 004: FSAE / Engineering Bridge Content

### Goal

Approach UNM FSAE respectfully as a student and future contributor, not as a brand trying to use their name.

### First moves

- request facility tour
- ask what skills to build before transferring
- ask where help is needed
- study FSAE design, cost, fabrication, testing, and presentation

### Later possible support

Only after permission and trust:

- student builder interviews
- sponsor highlight reels
- behind-the-scenes content
- part-of-the-week engineering breakdowns
- community education posts

## Proof Project 005: TSC One-Page Concept + Mini Deck

### Required before bigger outreach

TSC needs a clean proof packet before contacting big circles.

Minimum assets:

1. one-page TSC concept sheet
2. 10-slide mini deck
3. Culture Garage 001 event concept
4. sponsor opportunity list
5. safety and logistics overview
6. photo/reel proof package
7. clear ask: advice, collaboration, or support

## Proof package checklist

After each proof project, collect:

- date
- location
- who was involved
- what happened
- photos
- short reel
- founder notes
- sponsor/partner value created
- lessons learned
- next ask
- what can be shown publicly
- what must stay private

## Atlas feature requirements

Atlas should eventually support proof projects with:

- project status
- checklist
- contacts
- assets needed
- media folder links
- sponsor deliverables
- recap prompt generator
- follow-up reminders
- private/public flag

## Current priority

Start with:

1. Auto / Audio / Fabrication proof lane
2. Culture Garage 001 one-page concept
3. Academy micro-workshop outline
4. FSAE tour/learning request

Do not chase national brands until proof exists.
